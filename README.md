
# World edit

An Onix Client Plugin made for building and manipulating the world.


## New Features
- Noise (perlin)
- Bezier curves
- Circles on arbitrary planes
- Smoothing
- Undo and Redo
- Axiom like tenth slot selector
- Box command
- Extrude command
- Move command
- Arbitrary selection

## Changed things and issues
- Now uses axe instead of sword for selection
- Block states are not handled properly
- Noise patterns use `/` to delimit terms like `50%dirt` instead of `,`
